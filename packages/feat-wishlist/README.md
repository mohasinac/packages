# @mohasinac/feat-wishlist

> **Layer 5** — Wishlist feature module: wishlist button, wishlist page, and drawer.

## Install

```bash
npm install @mohasinac/feat-wishlist
```

---

## Components

```tsx
import { WishlistButton, WishlistPage, WishlistDrawer } from "@mohasinac/feat-wishlist";

<WishlistButton productId={product.id} />
<WishlistDrawer open={open} onClose={() => setOpen(false)} />
```
