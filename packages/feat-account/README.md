# @mohasinac/feat-account

> **Layer 5** — User account portal: profile page, address book, order history, payment methods, and notifications.

## Install

```bash
npm install @mohasinac/feat-account
```

Peer dependencies: React ≥ 18, Next.js ≥ 14, next-intl ≥ 3.

---

## Components

```tsx
import { ProfilePage, AddressBook, PaymentMethods, OrderHistory } from "@mohasinac/feat-account";
```
